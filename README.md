# بيكورا IPA

موقع لتحميل ألعاب iOS IPA.

## التثبيت

```bash
npm install
```

## التشغيل في وضع التطوير

```bash
npm run dev
```

## البناء للإنتاج

```bash
npm run build
```

ثم قم بخدمة محتوى مجلد `dist` عبر أي خادم ويب (Nginx، Apache، إلخ).

## إضافة الملفات

- **ملفات IPA**: ضعها في مجلد `public/ipa/`
  - `Pekora2017.ipa`
  - `Pekora2020.ipa`
  - `KoroneTwentyOne2.ipa`

- **صور الألعاب**: ضعها في مجلد `public/images/`
  - `pekora2017.jpg`
  - `pekora2020.jpg`
  - `pekora2021.jpg`
